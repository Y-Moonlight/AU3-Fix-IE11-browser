# AU3-Fix-IE11-browser
AU3-Fix IE11 browser，Win11开始内置IE11浏览器以无法打开，强制打开会跳转到edge浏览器，本工具可以修复IE11，使IE浏览器在Win11下可以正常运行

## 软件特点：
1.采用AU3代码编写，简单高效

2.可以修复 IE11，使IE浏览器在Win11下可以正常运行

3.GUI界面化，操作简单，傻瓜式功能
